# 아키텍처 구성도 발표 대본

## 발표 대본

이번 슬라이드에서는 InterviewMate AI의 전체 아키텍처를 설명드리겠습니다.

InterviewMate AI는 개발자 취업 준비생이 이력서 작성, 자기소개 생성, 기업 추천, 모의면접, 면접 리포트, 포트폴리오 생성까지 한 흐름에서 준비할 수 있도록 설계한 AI 기반 취업 준비 플랫폼입니다.

구성도는 크게 사용자, 프론트엔드, 백엔드 API, AI Core Platform, 외부 연동, 화상 분석, 결제 시스템으로 나누어 볼 수 있습니다.

먼저 사용자는 웹앱에 접속해 대시보드에서 기능을 선택합니다. 프론트엔드는 Next.js 기반으로 구성되어 있고, 대시보드, 이력서 생성 및 피드백, 1분 자기소개, AI 모의면접, 기업 추천, 멤버십 화면을 제공합니다. 사용자가 입력한 이력서, 기술 스택, 프로젝트 경험, 자기소개 내용은 프론트엔드에서 받아 백엔드 API로 전달됩니다.

백엔드 API는 서비스의 중간 처리 역할을 합니다. API Gateway를 통해 요청을 받고, 인증과 권한을 확인하며, 멤버십 등급에 따라 사용 가능한 기능을 제어합니다. 예를 들어 무료 사용자는 기본 기능 중심으로 이용하고, Premium이나 Pro 사용자는 모의면접, 표정/음성 분석, 합격 이력서 비교 같은 고급 기능까지 사용할 수 있도록 설계했습니다.

그다음 핵심 영역은 AI Core Platform입니다. 이 영역에는 이력서 생성, 이력서 첨삭, 자기소개 생성, 기업 추천, 답변 평가, 면접 질문 생성, 포트폴리오 생성, 합격 이력서 비교 기능이 모듈별로 분리되어 있습니다. 기능을 나누어 둔 이유는 각 기능마다 필요한 입력과 평가 기준이 다르기 때문입니다. 이력서 첨삭은 구조와 직무 적합성을 보고, 면접 답변 평가는 답변의 논리성, 구체성, 전달력을 보는 식으로 각각 다른 기준을 적용합니다.

외부 연동 영역에서는 OpenAI 또는 Gemini 같은 생성형 AI API와 연결하고, 개발자 취업 준비생에게 중요한 GitHub API 연동도 고려했습니다. GitHub API는 사용자의 프로젝트와 기술 스택을 분석해 이력서나 포트폴리오 생성에 활용할 수 있습니다. 또한 기업 추천 기능은 채용공고 수집 데이터와 연결해 사용자의 기술 스택에 맞는 기업이나 직무 방향을 추천하는 구조입니다.

면접 기능은 화상 분석 영역과 연결됩니다. 초기 프로토타입에서는 텍스트 기반 면접 피드백이나 기본 모의면접 흐름을 먼저 구현하고, 이후에는 웹캠과 마이크를 활용해 표정 분석, 음성 분석, 말 속도, 목소리 톤 같은 피드백으로 확장할 수 있습니다.

마지막으로 결제 시스템은 PG사 연동과 정기 구독 구조로 설계했습니다. 멤버십은 Free, Basic, Premium, Pro처럼 단계적으로 나누고, 등급에 따라 자기소개, 이력서 첨삭, AI 모의면접, 포트폴리오 생성, 합격 이력서 비교, 취업 경쟁력 분석 같은 기능 접근 범위를 다르게 가져갑니다.

정리하면, InterviewMate AI의 아키텍처는 사용자가 프론트엔드에서 취업 준비 기능을 이용하면 백엔드가 인증과 권한을 처리하고, AI Core Platform이 기능별 분석과 생성을 수행한 뒤 결과를 다시 리포트와 피드백 형태로 보여주는 구조입니다. 이 구조를 통해 초기에는 Google AI Studio 기반 프로토타입으로 빠르게 검증하고, 이후에는 로그인, 멤버십, 결제, GitHub 연동, 화상 면접 분석까지 단계적으로 확장할 수 있습니다.

## 짧은 버전

이 아키텍처는 프론트엔드, 백엔드 API, AI Core Platform, 외부 연동, 화상 분석, 결제 시스템으로 구성되어 있습니다.

사용자는 Next.js 기반 웹앱에서 이력서 생성, 자기소개, 기업 추천, 모의면접, 리포트 기능을 이용합니다. 요청은 백엔드 API로 전달되고, 백엔드는 인증과 권한, 멤버십 상태를 확인합니다. 이후 AI Core Platform에서 이력서 첨삭, 자기소개 생성, 기업 추천, 면접 질문 생성, 답변 평가 같은 기능을 모듈별로 처리합니다.

외부로는 생성형 AI API, GitHub API, 채용공고 수집 데이터와 연결되고, 향후 화상 분석 기능을 통해 표정과 음성 피드백까지 확장할 수 있습니다. 결제 시스템은 Free, Basic, Premium, Pro 등급별 기능 제공을 관리합니다.

즉, InterviewMate AI는 취업 준비생이 로그인 후 이력서, 자기소개, 기업 추천, 면접, 리포트, 포트폴리오까지 이어지는 취업 준비 과정을 하나의 서비스 흐름으로 경험할 수 있도록 설계한 구조입니다.

## 슬라이드에서 짚을 순서

1. 가장 위의 사용자 영역을 보며 취업 준비생이 서비스에 접속하는 지점이라고 설명한다.
2. 왼쪽 프론트엔드 영역을 보며 대시보드, 이력서, 자기소개, 모의면접, 기업 추천, 멤버십 화면을 설명한다.
3. 가운데 백엔드 API 영역을 보며 API Gateway, 인증/권한, 구독 관리 역할을 설명한다.
4. 오른쪽 AI Core Platform을 보며 이력서 생성, 첨삭, 자기소개 생성, 기업 추천, 답변 평가, 면접 질문 생성, 포트폴리오 생성 기능을 설명한다.
5. 아래쪽 외부 연동 영역을 보며 생성형 AI API, GitHub API, 채용공고 수집 데이터를 설명한다.
6. 화상 분석 영역을 보며 향후 표정 분석과 음성 분석으로 면접 피드백을 확장한다고 설명한다.
7. 결제 시스템과 멤버십 다이어그램을 연결해 Free, Basic, Premium, Pro 등급별 기능 차이를 설명한다.
8. 사용자 흐름 다이어그램을 보조로 활용해 로그인부터 포트폴리오 생성까지 이어지는 실제 이용 순서를 설명한다.
